# Front-Size Plots

Generated by `scripts/plot_front_stats.py`.

The histogram buckets use powers of two so the small-front region is visible:
`<=4`, `5-8`, `9-16`, `17-32`, `33-64`, `65-128`, `129-256`, `>256`.
