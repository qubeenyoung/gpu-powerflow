# Linear Solver Benchmark - 2026-05-29

## Setup

- Cases: 78 Jacobian dumps from `/datasets/matpower_linear_systems`
- Repeats: 10 independent process runs per case/backend
- Inner repeat: 5 factorize/solve calls per process; runner reports the median
- Backends: cuDSS FP64 and custom FP64
- cuDSS matching: True
- cuDSS multithreading layer: True
- cuDSS threading lib: `runner default`
- cuDSS host nthreads: runner default
- Input per case: `J.mtx` and `F.mtx` generated from MATPOWER Newton-Raphson Jacobian data
- Timed phases: read, upload, setup/set_data, analyze, factorize, solve, download/residual check

## Artifacts

- `raw_runs.csv`: every measured process run
- `summary.csv`: per-case/backend aggregate
- `comparison.csv`: cuDSS vs custom ratios per case
- `metadata.json`: environment and command metadata
